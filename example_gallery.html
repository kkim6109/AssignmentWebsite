The picture above is my own version of Tetris that I have created. It was a fun project that was a lot harder than I thought it would be. The blocks, grid, and anything with a gray background were drawn with code rather than using a default element. And I took the privilege of trying to make it look nice rather than it being a completely boring program, it even has music! I still play this game every once in a while when I have a bit of time in my schedule.

This is another game that I have created except with some more creativity added in how the game works, rather than only recreating the original game, I also added my own difficulties. This is snake, the normal difficulty is simply the original game. The easy difficulty however has a new twist on the game where you can go through the walls and appear on the opposite wall. The hard difficulty is like the normal difficulty, but walls appear on screen as you pick up more apples. And not only was this a program where I got to try out new creative ideas, this program was also the first program I created where I programatically generated the visuals rather than using an engine or using images.
